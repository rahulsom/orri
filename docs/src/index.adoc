== Summary

Each worksheet becomes a table.
The first row in each worksheet becomes the column header.
The remaining rows become table data.
Google Sheets filter views are exposed as JDBC views.
Boolean cells can come from either literal `true` and `false` values or checkbox-backed cells.
Connections are writable by default for worksheet tables.

If you want to force a read-only connection, set `readOnly=true` in the JDBC URL or connection properties.

== Type Mapping

Columns are inferred from the worksheet contents.

* Boolean columns map to JDBC booleans.
* Numeric columns map to JDBC decimals.
* All other columns map to JDBC strings.
* If a checkbox column is used in Google Sheets, the driver exposes it as a boolean column when the rendered values are `true` and `false`.
